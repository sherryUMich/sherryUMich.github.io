Juxtaposed Public Program

February 08, 2016

ARCHITECTURE, ACADEMIC PROJECT

Undergraduate 2016.2 - 2016.6, UNNC

INDIVIDUAL WORK

Project Location: SHANGHAI


The focus of the project is to design a public architecture based on the masterplan of livable, human scale, resource efficient cities.
The concept of this design is “SUPERIMPOSITION” which is based on BERNARD TSCHUMI’s theory of collision by overlapping the needs of different groups of users to generate a dialogue between people, event and space. Besides, the concept fully considers the site condition which is a juxtaposed area with residential blocks, office blocks and green area around.
